# Entidades 

Local na qual ficam todas as entidades do projeto

## Tipod de Função da Entidade Body
Variavel na qual vai emitir qual tipo de função ela irá execurtar ao enviar ou receber.

0 - Descontectado

1 - Conectado

2 - Conectado para lista

3 - Nova conexão 

4 - Inativo

5 - Excluir mensagem
